**Project Title:** Cryptography Algorithms Implementation

**Objective:**
To implement SHA-256 hashing using Python for secure password protection.

**Tools Used:**

* Python
* VS Code

**Description:**
This project converts a normal password into a hashed value using the SHA-256 cryptographic algorithm. Hashing improves security by protecting sensitive information.

**Outcome:**
Learned basics of cryptography, hashing, and password security.
